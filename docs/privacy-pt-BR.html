Política de Privacidade

Este aplicativo "VidEncoBox" pode coletar e usar certas informações para melhorar a experiência do usuário e exibir anúncios.

1. Informações coletadas

  • Dados de uso do aplicativo (contagem de inicializações, logs de erros, etc.)
  • Identificador de publicidade (IDFA)
  • Arquivos de mídia (vídeo, áudio, imagens) - armazenados apenas no seu dispositivo
  • Dados de projeto (nome, descrição, tags, etc.) - armazenados apenas no seu dispositivo
  • Histórico de navegação e downloads - armazenado apenas no seu dispositivo
  • Favoritos e lista de leitura - armazenados apenas no seu dispositivo

2. Finalidade do uso

  • Edição, conversão e codificação de arquivos de mídia
  • Navegação de conteúdo web e download de mídia
  • Exibição de anúncios (usando Google AdMob)
  • Análise de falhas e erros

3. Transmissão de dados externos

Este aplicativo usa os seguintes serviços externos:

  • Google AdMob (publicidade)
  • Google User Messaging Platform (gestão de consentimento GDPR/CCPA)
  • Apple App Store (pagamentos de assinatura)
  • SKAdNetwork (atribuição de publicidade)

4. Permissão de rastreamento

Um popup de permissão de rastreamento (App Tracking Transparency) pode aparecer quando o aplicativo é iniciado para fins publicitários. Anúncios personalizados não serão exibidos sem a permissão do usuário.

5. Navegador integrado

Este aplicativo inclui um navegador integrado para navegar conteúdo web e baixar mídia.

  • Histórico de navegação, histórico de downloads e favoritos são todos armazenados apenas no seu dispositivo
  • Cookies obtidos de sites são usados apenas para processamento de downloads e armazenados no seu dispositivo
  • Nenhum histórico de navegação é transmitido para servidores externos

6. Armazenamento de dados e segurança

Os dados criados e armazenados por este aplicativo são geralmente armazenados apenas no seu dispositivo.

  • Arquivos de mídia e dados de projeto: armazenados no armazenamento local do seu dispositivo
  • Informações sensíveis: criptografadas e armazenadas usando Keychain
  • Sincronização iCloud: Os dados do projeto são sincronizados com o iCloud apenas quando habilitado pelo usuário
  • Bloqueio de pasta: Proteção usando autenticação biométrica (Face ID/Touch ID) disponível

7. Permissões do dispositivo

Este aplicativo pode usar as seguintes permissões do dispositivo:

  • Câmera: Usada para digitalização de códigos QR
  • Biblioteca de fotos: Usada para salvar e carregar capturas de tela e mídia

8. Assinaturas

Este aplicativo oferece compras no aplicativo (assinaturas). O processamento de pagamentos é feito pelo sistema de pagamento da App Store da Apple, e não coletamos nem armazenamos informações de pagamento (como números de cartão de crédito). Membros Premium terão todos os anúncios removidos.

9. Exclusão de dados

Os dados criados pelo usuário (arquivos de mídia, projetos, histórico, etc.) podem ser excluídos diretamente no aplicativo. Desinstalar o aplicativo excluirá todos os dados armazenados no seu dispositivo.
