Kebijakan Privasi

Aplikasi ini "VidEncoBox" dapat mengumpulkan dan menggunakan informasi tertentu untuk meningkatkan pengalaman pengguna dan menampilkan iklan.

1. Informasi yang Dikumpulkan

  • Data penggunaan aplikasi (jumlah peluncuran, log kesalahan, dll.)
  • Pengidentifikasi iklan (IDFA)
  • File media (video, audio, gambar) - disimpan hanya di perangkat Anda
  • Data proyek (nama, deskripsi, tag, dll.) - disimpan hanya di perangkat Anda
  • Riwayat penelusuran dan unduhan - disimpan hanya di perangkat Anda
  • Bookmark dan daftar bacaan - disimpan hanya di perangkat Anda

2. Tujuan Penggunaan

  • Mengedit, mengonversi, dan mengkodekan file media
  • Menelusuri konten web dan mengunduh media
  • Tampilan iklan (menggunakan Google AdMob)
  • Analisis kerusakan dan kesalahan

3. Transmisi Data Eksternal

Aplikasi ini menggunakan layanan eksternal berikut:

  • Google AdMob (periklanan)
  • Google User Messaging Platform (manajemen persetujuan GDPR/CCPA)
  • Apple App Store (pembayaran langganan)
  • SKAdNetwork (atribusi iklan)

4. Izin Pelacakan

Saat aplikasi diluncurkan, popup izin pelacakan (App Tracking Transparency) mungkin muncul untuk tujuan periklanan. Iklan yang dipersonalisasi tidak akan ditampilkan tanpa izin pengguna.

5. Browser Bawaan

Aplikasi ini menyertakan browser bawaan untuk menelusuri konten web dan mengunduh media.

  • Riwayat penelusuran, riwayat unduhan, dan bookmark semuanya disimpan hanya di perangkat Anda
  • Cookie yang diperoleh dari situs web hanya digunakan untuk pemrosesan unduhan dan disimpan di perangkat Anda
  • Tidak ada riwayat penelusuran yang dikirim ke server eksternal

6. Penyimpanan Data dan Keamanan

Data yang dibuat dan disimpan oleh aplikasi ini umumnya disimpan hanya di perangkat Anda.

  • File media dan data proyek: disimpan di penyimpanan lokal perangkat Anda
  • Informasi sensitif: dienkripsi dan disimpan menggunakan Keychain
  • Sinkronisasi iCloud: Data proyek disinkronkan dengan iCloud hanya ketika diaktifkan oleh pengguna
  • Kunci folder: Perlindungan menggunakan autentikasi biometrik (Face ID/Touch ID) tersedia

7. Izin Perangkat

Aplikasi ini dapat menggunakan izin perangkat berikut:

  • Kamera: Digunakan untuk memindai kode QR
  • Perpustakaan Foto: Digunakan untuk menyimpan dan memuat tangkapan layar dan media

8. Langganan

Aplikasi ini menawarkan pembelian dalam aplikasi (langganan). Pemrosesan pembayaran dilakukan melalui sistem pembayaran Apple App Store, dan kami tidak mengumpulkan atau menyimpan informasi pembayaran (seperti nomor kartu kredit). Anggota Premium akan menghapus semua iklan.

9. Penghapusan Data

Data yang dibuat pengguna (file media, proyek, riwayat, dll.) dapat dihapus langsung di dalam aplikasi. Menghapus instalasi aplikasi akan menghapus semua data yang tersimpan di perangkat Anda.
