Dasar Privasi

Aplikasi ini "VidEncoBox" mungkin mengumpul dan menggunakan maklumat tertentu untuk meningkatkan pengalaman pengguna dan memaparkan iklan.

1. Maklumat yang Dikumpul

  • Data penggunaan aplikasi (bilangan pelancaran, log ralat, dll.)
  • Pengenal iklan (IDFA)
  • Fail media (video, audio, imej) - disimpan hanya pada peranti anda
  • Data projek (nama, penerangan, tag, dll.) - disimpan hanya pada peranti anda
  • Sejarah pelayaran dan muat turun - disimpan hanya pada peranti anda
  • Penanda buku dan senarai bacaan - disimpan hanya pada peranti anda

2. Tujuan Penggunaan

  • Menyunting, menukar dan mengekod fail media
  • Melayari kandungan web dan memuat turun media
  • Paparan iklan (menggunakan Google AdMob)
  • Analisis ranap dan ralat

3. Penghantaran Data Luaran

Aplikasi ini menggunakan perkhidmatan luaran berikut:

  • Google AdMob (pengiklanan)
  • Google User Messaging Platform (pengurusan persetujuan GDPR/CCPA)
  • Apple App Store (pembayaran langganan)
  • SKAdNetwork (atribusi iklan)

4. Kebenaran Penjejakan

Semasa melancarkan aplikasi, pop timbul kebenaran penjejakan (App Tracking Transparency) mungkin muncul untuk tujuan pengiklanan. Iklan yang diperibadikan tidak akan dipaparkan tanpa kebenaran pengguna.

5. Pelayar Terbina Dalam

Aplikasi ini termasuk pelayar terbina dalam untuk melayari kandungan web dan memuat turun media.

  • Sejarah pelayaran, sejarah muat turun dan penanda buku semuanya disimpan hanya pada peranti anda
  • Kuki yang diperoleh daripada laman web digunakan hanya untuk pemprosesan muat turun dan disimpan pada peranti anda
  • Tiada sejarah pelayaran dihantar ke pelayan luaran

6. Penyimpanan Data dan Keselamatan

Data yang dicipta dan disimpan oleh aplikasi ini secara umumnya disimpan hanya pada peranti anda.

  • Fail media dan data projek: disimpan dalam storan tempatan peranti anda
  • Maklumat sensitif: disulitkan dan disimpan menggunakan Keychain
  • Penyegerakan iCloud: Data projek disegerakkan dengan iCloud hanya apabila diaktifkan oleh pengguna
  • Kunci folder: Perlindungan menggunakan pengesahan biometrik (Face ID/Touch ID) tersedia

7. Kebenaran Peranti

Aplikasi ini mungkin menggunakan kebenaran peranti berikut:

  • Kamera: Digunakan untuk mengimbas kod QR
  • Pustaka Foto: Digunakan untuk menyimpan dan memuatkan tangkapan skrin dan media

8. Langganan

Aplikasi ini menawarkan pembelian dalam aplikasi (langganan). Pemprosesan pembayaran dikendalikan melalui sistem pembayaran Apple App Store, dan kami tidak mengumpul atau menyimpan maklumat pembayaran (seperti nombor kad kredit). Ahli Premium akan mempunyai semua iklan dibuang.

9. Pemadaman Data

Data yang dicipta oleh pengguna (fail media, projek, sejarah, dll.) boleh dipadamkan secara langsung dalam aplikasi. Menyahpasang aplikasi akan memadamkan semua data yang disimpan pada peranti anda.
