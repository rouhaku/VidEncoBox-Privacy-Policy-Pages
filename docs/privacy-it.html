Informativa sulla privacy

Questa app "VidEncoBox" può raccogliere e utilizzare determinate informazioni per migliorare l'esperienza utente e visualizzare annunci pubblicitari.

1. Informazioni raccolte

  • Dati di utilizzo dell'app (numero di avvii, log degli errori, ecc.)
  • Identificatore pubblicitario (IDFA)
  • File multimediali (video, audio, immagini) - memorizzati solo sul dispositivo
  • Dati di progetto (nome, descrizione, tag, ecc.) - memorizzati solo sul dispositivo
  • Cronologia di navigazione e download - memorizzata solo sul dispositivo
  • Segnalibri e lista di lettura - memorizzati solo sul dispositivo

2. Scopo dell'utilizzo

  • Modifica, conversione e codifica di file multimediali
  • Navigazione di contenuti web e download di media
  • Visualizzazione di annunci (utilizzando Google AdMob)
  • Analisi di crash ed errori

3. Trasmissione dati esterni

Questa app utilizza i seguenti servizi esterni:

  • Google AdMob (pubblicità)
  • Google User Messaging Platform (gestione del consenso GDPR/CCPA)
  • Apple App Store (pagamenti abbonamento)
  • SKAdNetwork (attribuzione pubblicitaria)

4. Autorizzazione al tracciamento

All'avvio dell'app potrebbe apparire un popup di autorizzazione al tracciamento (App Tracking Transparency) per scopi pubblicitari. Gli annunci personalizzati non verranno visualizzati senza l'autorizzazione dell'utente.

5. Browser integrato

Questa app include un browser integrato per navigare contenuti web e scaricare media.

  • Cronologia di navigazione, cronologia download e segnalibri sono tutti memorizzati solo sul dispositivo
  • I cookie ottenuti dai siti web sono utilizzati solo per l'elaborazione dei download e memorizzati sul dispositivo
  • Nessuna cronologia di navigazione viene trasmessa a server esterni

6. Archiviazione dati e sicurezza

I dati creati e archiviati da questa app sono generalmente memorizzati solo sul dispositivo.

  • File multimediali e dati di progetto: memorizzati nella memoria locale del dispositivo
  • Informazioni sensibili: crittografate e memorizzate usando Keychain
  • Sincronizzazione iCloud: I dati di progetto vengono sincronizzati con iCloud solo se abilitato dall'utente
  • Blocco cartelle: Protezione tramite autenticazione biometrica (Face ID/Touch ID) disponibile

7. Autorizzazioni dispositivo

Questa app può utilizzare le seguenti autorizzazioni del dispositivo:

  • Fotocamera: Utilizzata per la scansione di codici QR
  • Libreria foto: Utilizzata per salvare e caricare screenshot e media

8. Abbonamenti

Questa app offre acquisti in-app (abbonamenti). L'elaborazione dei pagamenti avviene tramite il sistema di pagamento dell'App Store di Apple e non raccogliamo né memorizziamo informazioni di pagamento (come numeri di carta di credito). I membri Premium avranno tutti gli annunci rimossi.

9. Eliminazione dati

I dati creati dall'utente (file multimediali, progetti, cronologia, ecc.) possono essere eliminati direttamente nell'app. La disinstallazione dell'app eliminerà tutti i dati memorizzati sul dispositivo.
