Политика конфиденциальности

Это приложение «VidEncoBox» может собирать и использовать определенную информацию для улучшения пользовательского опыта и показа рекламы.

1. Собираемая информация

  • Данные об использовании приложения (количество запусков, журналы ошибок и т.д.)
  • Рекламный идентификатор (IDFA)
  • Медиафайлы (видео, аудио, изображения) - хранятся только на вашем устройстве
  • Данные проекта (название, описание, теги и т.д.) - хранятся только на вашем устройстве
  • История просмотров и загрузок - хранится только на вашем устройстве
  • Закладки и список чтения - хранятся только на вашем устройстве

2. Цель использования

  • Редактирование, конвертация и кодирование медиафайлов
  • Просмотр веб-контента и загрузка медиа
  • Показ рекламы (с использованием Google AdMob)
  • Анализ сбоев и ошибок

3. Внешняя передача данных

Это приложение использует следующие внешние сервисы:

  • Google AdMob (реклама)
  • Google User Messaging Platform (управление согласием GDPR/CCPA)
  • Apple App Store (платежи за подписку)
  • SKAdNetwork (атрибуция рекламы)

4. Разрешение на отслеживание

При запуске приложения может появиться всплывающее окно с запросом разрешения на отслеживание (App Tracking Transparency) для рекламных целей. Персонализированная реклама не будет показана без разрешения пользователя.

5. Встроенный браузер

Это приложение включает встроенный браузер для просмотра веб-контента и загрузки медиа.

  • История просмотров, история загрузок и закладки хранятся только на вашем устройстве
  • Файлы cookie, полученные с веб-сайтов, используются только для обработки загрузок и хранятся на вашем устройстве
  • История просмотров не передается на внешние серверы

6. Хранение данных и безопасность

Данные, созданные и сохраненные этим приложением, как правило, хранятся только на вашем устройстве.

  • Медиафайлы и данные проекта: хранятся в локальном хранилище вашего устройства
  • Конфиденциальная информация: зашифрована и хранится с использованием Keychain
  • Синхронизация iCloud: Данные проекта синхронизируются с iCloud только при включении пользователем
  • Блокировка папок: Доступна защита с использованием биометрической аутентификации (Face ID/Touch ID)

7. Разрешения устройства

Это приложение может использовать следующие разрешения устройства:

  • Камера: Используется для сканирования QR-кодов
  • Библиотека фотографий: Используется для сохранения и загрузки скриншотов и медиа

8. Подписки

Это приложение предлагает покупки в приложении (подписки). Обработка платежей осуществляется через платежную систему Apple App Store, и мы не собираем и не храним платежную информацию (например, номера кредитных карт). У премиум-пользователей будет удалена вся реклама.

9. Удаление данных

Данные, созданные пользователем (медиафайлы, проекты, история и т.д.), можно удалить непосредственно в приложении. Удаление приложения удалит все данные, хранящиеся на вашем устройстве.
